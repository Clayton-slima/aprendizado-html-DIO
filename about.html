Estrutura Básica do HTML

(Voltar)

A estrutura básica do HTML consiste em um conjunto de elementos que define a estrutura e o conteúdo de uma página web. Veremos abaixo uma descrição geral dos principais elementos que compõem a estrutura básica do HTML:

Tag <HTML>: A tag <HTML> é o elemento raiz de um documento HTML e envolve todo o conteúdo da página. Ela indica que o documento é escrito em HTML e define o escopo do documento.

Tag <head>: A tag <head> é um elemento filho direto da tag <HTML>. Ela contém informações sobre o documento HTML que não são diretamente exibidas na página, mas fornecem metadados, como o título da página, links para folhas de estilo CSS, scripts JavaScript e outros elementos relacionados ao cabeçalho da página.

Tag <title>: A tag <title> é um elemento filho direto da tag <head>. Ela define o título da página, que é exibido na barra de título do navegador ou na guia do navegador.

Tag <body>: A tag <body> é um elemento filho direto da tag <HTML>. Ela contém todo o conteúdo visível da página, como texto, imagens, links, tabelas e outros elementos HTML. É dentro da tag <body> que os usuários veem o conteúdo principal da página.

Dentro da tag <body>, você pode usar uma variedade de tags HTML para estruturar e formatar o conteúdo da página. Por exemplo, as tags <h1> a <h6> são usadas para cabeçalhos de diferentes níveis, as tags <p> para parágrafos, <a> para links, <img> para imagens, <ul> / <ol> com <li> para listas não ordenadas e ordenadas, respectivamente, e assim por diante.

Links

(Voltar)

Links são elementos fundamentais em HTML para permitir a navegação entre páginas e recursos na web. Eles são criados usando a tag <a> (abreviação de anchor).

Aqui está um exemplo básico de como criar um link em HTML:

DIO.me (<a href="https://www.dio.me">DIO.me</a>)

No exemplo acima, o atributo href é usado para especificar o destino do link, que pode ser um URL completo, um caminho relativo ou um identificador de âncora dentro da mesma página.

O texto dentro da tag <a> será exibido como o texto do link no navegador. Quando o usuário clicar no link, ele será redirecionado para o URL especificado no atributo href.

Além do atributo href, existem outros atributos úteis que podem ser usados com a tag <a>:

Target: Define como o link deve ser aberto. Por exemplo, target="_blank" abrirá o link em uma nova janela ou guia do navegador.

Title: Define um texto de dica que aparece quando o usuário passa o mouse sobre o link.

Rel: Define a relação entre a página atual e o link, como "nofollow" para informar que os mecanismos de pesquisa não devem seguir esse link.

DIO.me (<a href="https://www.dio.me" target="_blank" title="Visite a melhor plataforma de ensino online!" rel="nofollow">DIO.me</a>)
